## Сценарии NPM

Cобираем проект в режиме разработчика
```js
npm run dev
```
---
Собираем проект 
```js
npm run build
```
---
Создаем архив проекта 
```
npm run zip
```
---
Заливаем проект на сервер, все настройки в файле gulp/config/ftp.js 
```
npm run deploy
```
---
Создаем спрайт из иконок
```
npm run svgSprive
```
